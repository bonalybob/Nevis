Nevis

View Source on Github View Latest Release

A Open Source, Easy to Use, Fast Download and Results program for the SportIdent Timing system.

Features

With Nevis you can:

  • Add + Edit Entries
  • View of Entries Live with Searching and Sorting
  • Import Entries from IOF XML
  • Download SI Cards (Tested with SI Card 5, 8, 10, 11 + SIAC) (SI Card 6 Support to Come Later)
  • Use Automatic Course Checking
  • Add + Edit Courses
  • Import Courses from IOF XML or JSON
  • View Results
  • Export Results in PDF, HTML or IOF XML(Partially)
  • Have all data Encrypted in Database
  • Splits Printing
  • SI Card Archive Lookup

Install

Get the Windows installer from Releases page

OR To build from source:

git clone https://github.com/bonalybob/nevis.git
npm install
npm run styles
npm start

To create your own executable then run:

npm run dist
